\doxysubsection{Подробное описание}
Класс для хранения хранящихся в блоке данных в произвольном формате. 

Единственное, за что отвечает класс \mbox{\hyperlink{class_block_chain_core_1_1_block_contained_data}{Block\+Contained\+Data}} -\/ это хранения и сериализация данных. Данные, хранимые в нем, абсолютно произвольны. За валидацию хранимых данных отвечают внешние ресурсы, никакого отношения к ядру абсолютно не имеющие.
