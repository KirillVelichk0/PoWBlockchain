Подробное описание

Данный класс полностью повторяет функционал класса boost::archive::text_oarchive. Его единственная задача - обеспечить дополнительный сценарий сериализации для верификации блока. Достигается это путем перегрузки шаблона serialize в структуре BlockHashInfo.
